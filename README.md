📝 Project Description

I prompted a new AI tool to design a complete schematic for an **100W PD power bank **.

The AI managed to generate:
- Proper symbol selection
- read datasheet.
- draw schematics.
- pack full project files including:kicad_sch,kicad_pro,libary

💾 How to View

1. Download the [100W powerbank].zip file located above in this repository.
2. Extract the contents.
3. Open the .kicad_pro file directly in KiCad 8.0 or later.

Feel free to check the schematics and share your findings on the KiCad forum!
